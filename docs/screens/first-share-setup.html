Edge Cases
  • Tone already configured for this platform: Skip platform + tone selection, go straight to generated result
  • 3 tones max per platform: 4th tone greyed out with tooltip "Max 3 voix pour une marque forte"
  • Brainstorming skipped: AI generates with tones only (lower quality but functional)
  • Brainstorming already done: Brand voice callout hidden, results directly use saved brand voice
  • No internet connection: Show error state, retry button
  • Generation takes too long (>10s): Show timeout message with "Try again" option
  • Platform not installed: Share Sheet shows available sharing options regardless
Acceptance Criteria
  • Sheet modal (not full screen) with backdrop blur and swipe-to-dismiss
  • Platform bento grid shows 5 platforms (Instagram, LinkedIn, X, TikTok, Pinterest)
  • Tone cards show same phrase in 5 different voices for comparison
  • Multi-select 1-3 tones max per platform
  • Brainstorming trigger optional via callout card with CTA
  • Brainstorming uses stacked Q&A card design (reuses existing pattern)
  • Generated result shows editable hook + caption + hashtags
  • "Publier via [platform]" opens native Share Sheet
  • If tone already configured, skip directly to generation
  • All state persists: platform + tone saved for future shares
