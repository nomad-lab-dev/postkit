Edge Cases
  • No pillars created: Show "Create your first pillar" CTA instead of grid
  • All pillars fully reviewed: Celebration state "All caught up!" with confetti emoji
  • Scan running: Counts update in real-time with "+" suffix to indicate more coming
  • Pillar with 0 to review: Green checkmark icon and "All reviewed!" text
  • Very many pillars (10+): Grid scrolls vertically, all pillars accessible
Acceptance Criteria
  • Bento grid (2 columns) with emoji icon, pillar name, and mini thumbnail grid per card
  • Quantity overlay badge shows number of assets to review per pillar
  • Tap pillar card enters swipe card mode for that pillar
  • Pillar with 0 items shows green checkmark "All reviewed!"
  • Live count updates during active scan
  • No pillars state shows creation CTA
  • 3-second rule: selector loads within 3 seconds including thumbnails
