Edge Cases
  • Video asset: Show video player with controls. First frame displayed as thumbnail. Duration badge.
  • Not classified: Show "Classify Now" button instead of metadata fields
  • Deleted photo: Error state with cleanup option to remove orphaned record from PillarBox
  • Restore from non-pertinent: Action available in non-pertinent gallery, asset returns to active grid
  • All pillars removed: Asset still exists but is untagged, visible in "no pillar" filter
  • Very long tag list: Tags wrap to multiple lines, scrollable within the chips area
Acceptance Criteria
  • Sheet slides up from bottom with swipe-to-dismiss
  • All metadata fields editable inline (pillars, tags, location, cadrage)
  • Auto-save on every edit (no manual save button needed)
  • "Mark non-pertinent" moves asset out of grid and suggestions
  • Swipe left/right navigates between assets in current grid context
  • Video playback works with standard controls
  • 3 states clearly distinguished: Active, Non-pertinent, Used
  • "Classify Now" available for unclassified assets
