Edge Cases
  • Pre-filled by scan: 3-4 pillars pre-toggled ON with "Detecte par l'IA" badge. User can toggle off AI suggestions and add their own.
  • No scan results (0 piliers detectes): All pillars shown in OFF state, user selects manually
  • Duplicate custom name: inline error "This pillar already exists" shown inside partial sheet, "Add" button disabled
  • Empty name in sheet: "Add Pillar" button disabled until name is entered
  • Long name (>30 chars): truncate in card with ellipsis, full name stored
  • 11+ pillars: nudge banner shown, but doesn't block
  • Back then Forward: selections preserved in ViewModel
  • AI emoji suggestion fails (network): default to star emoji ⭐
  • Special characters in name: allowed, whitespace trimmed
Acceptance Criteria
  • Step dots show 3/3 (2 done + 1 active)
  • Bento grid 2 columns with emoji + name + brief description per card
  • AI-detected pillars pre-toggled ON with "Detecte par l'IA" purple badge
  • Toggle on/off with clear visual feedback (border color + background)
  • Custom pillar: partial sheet with name field, emoji picker, color picker, and "Add" button
  • Emoji auto-suggest on textfield exit inside sheet (micro-animation)
  • Native emoji picker opens on emoji tap inside sheet
  • Color picker with 6+ swatches in sheet
  • Minimum 1 pillar enforced (Next disabled at 0)
  • > 10 pillars shows nudge banner
  • Duplicate detection with inline error
  • Nav buttons in same fixed position as all other onboarding steps
  • "Next" leads to Template Nudge (not Platforms/Tone)
  • "Back" leads to Quick Scan Preview (Step 2)
