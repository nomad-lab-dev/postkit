Edge Cases
  • 0 platforms selected on 3.1 → skip all tone sub-steps, go to Step 4
  • 1 platform selected → only 1 tone sub-step, badge shows "1/1"
  • 0 tones for a platform → allowed, AI defaults to Professional
  • 3 tones selected → 4th and 5th cards disabled with visual feedback + tooltip
  • Back from 3.3 → 3.2 → selections preserved for each platform
  • Large Dynamic Type → tone cards scroll vertically, descriptions may wrap
  • "Mix" option removed (v0.1 decision — too vague, replaced by multi-select)
  • Same example phrase across all 5 tones for direct comparison
Acceptance Criteria
  • Platform bento grid renders with toggle on/off visual feedback
  • 0 platforms → tone sub-steps skipped entirely
  • Sub-progress badges show which platform (with checkmark for completed ones)
  • Same phrase shown rewritten in 5 different tones per platform
  • Multi-select 1-3 enforced — tooltip at 3, 4th/5th greyed out
  • 0 tones for a platform is allowed (AI defaults to Professional)
  • Back between sub-steps preserves previous platform selections
  • Nav buttons (Back/Next) always in same fixed position across all sub-steps
  • "Mix" option is NOT present
  • Step dots show 3/5 for all sub-steps (main progress, not sub-progress)
  • Accessible: VoiceOver announces platform name, tone name, selection state
