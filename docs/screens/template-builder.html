Edge Cases
  • 0 slots: Cannot save template. "Add at least 1 slot" message shown, Save button disabled.
  • Duplicate template name: Allowed (templates have unique IDs, not unique names)
  • Very many slots (10+): Scrollable list, no hard limit. Preview grid adapts.
  • Last slot deleted: Confirmation "This is the last slot. Remove it?" to prevent accidental empty template.
  • Template in use by posts: Deleting template doesn't affect existing posts (they keep their slot structure).
Acceptance Criteria
  • Full CRUD for templates (create, read, update, delete)
  • Per-slot configuration: cadrage picker, pillar multi-select, tags, description
  • Drag handles to reorder slots, order persisted
  • Preview visual shows mini slot boxes with cadrage labels
  • Templates accessible from "Create Post" flow as starting point
  • 0 slots prevents save with clear feedback
  • Delete template requires confirmation
