Edge Cases
  • Empty bio: allowed. AI uses pillars + per-platform tones as fallback.
  • Platform with 0 tones: allowed. AI defaults to Professional for that platform.
  • 3 tones per platform: max enforced, 4th/5th greyed out with tooltip.
  • All platforms removed: allowed. AI generates generic content.
  • Brand Voice not done: Purple-bordered CTA "Affine ta brand voice (5-10 min)" shown. AI generates with tones only.
  • Brand Voice done: Green-bordered summary with personality, differentiator, values. "Refaire →" link to redo.
  • Redo onboarding: pre-fills all steps with current values. No data loss.
  • Rapid edits: auto-save debounced at 500ms.
  • Bio over 280 chars: counter red, stops accepting.
Acceptance Criteria
  • Section order: Bio → Platforms & Tones → Pillars → Brand Voice → Redo Onboarding
  • Bio editable inline with char counter
  • NO global tone picker — tone is per-platform only
  • Each platform card shows its assigned tones as badges
  • Tap platform → tone detail with same-phrase examples
  • Tone multi-select 1-3 max per platform
  • Brand Voice section: CTA if brainstorming not done, summary if done
  • Brand Voice summary shows personality, differentiator, values with green border
  • "Refaire →" link available on completed brand voice to redo brainstorming
  • AI preview reflects per-platform tone assignments + brand voice if available
  • "Manage Pillars" navigates correctly
  • "Redo Onboarding" with confirmation, pre-fills values
  • Auto-save on all changes
