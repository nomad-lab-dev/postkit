Edge Cases
  • 15K+ assets: LazyVGrid with pagination (50 per page), scroll-to-load for smooth performance
  • Zero filter results: Friendly empty state with "No assets found. Try different filters."
  • Deleted photo in library: Placeholder stripe pattern + "Clean up?" link for database cleanup
  • Multi-select + tab switch: Automatically exits multi-select mode, deselects all
  • Scan running: New assets appear in real-time with fade-in animation as they are classified
  • Location picker with no EXIF data: "No location data available" message in picker
Acceptance Criteria
  • All 6 filter types functional: pillar, cadrage, location, moment, media, status
  • Location picker auto-populated from EXIF city data across the library
  • Grid renders smoothly with 1000+ items (LazyVGrid + pagination)
  • Rich cards show thumbnail + pillar badge + cadrage icon
  • Multi-select entry: "Select" button top-right OR long press on card
  • Multi-select shows 2 action buttons (Create Post, Add to Existing)
  • Search bar filters by tag, location, and description
  • 3-second rule: grid loads and scrolls within 3 seconds
  • 3-column layout on iPhone, 4-5 columns on Mac
