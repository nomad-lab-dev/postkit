Edge Cases
  • No template: Free-form mode, "+" button to add slots manually without cadrage/pillar constraints
  • Slot requirements match 0 assets: Explorer shows empty state with editable filters to broaden search
  • All assets removed from post: Cannot publish (minimum 1 asset required)
  • Export album already exists: Update existing album, don't create duplicate
  • Photo library permission denied for export: Show Settings link to enable access
  • Template with many slots (8+): Slot grid scrolls vertically
Acceptance Criteria
  • Slots show template requirements (cadrage, pillar) when empty, thumbnail when filled
  • Contextual explorer opens pre-filtered by slot requirements
  • Explorer filters are editable (user can broaden or narrow)
  • Tags/location visible on filled slots as small overlay
  • Publish/Cancel flow works correctly with appropriate confirmations
  • Export to native Photos album works
  • Auto-save on all edits (name, description, slot changes)
  • Free-form mode available when no template is attached
