Design Decisions

  • 2 permissions only — Location removed. EXIF GPS data is read from photo metadata via PhotoKit, no Location Services permission needed.
  • Network auto-detected — Not a "permission" per se, but shown as a system check. If offline, offer "scan offline" (Core ML only, less accurate).
  • In most cases, 1 tap — Network is auto-green. User taps Photos once → both green → Launch enabled. Minimal friction.
  • Unlock card flip animation — Locked (grey, dashed, padlock) → granted (green, solid, checkmark + real data like photo count). Satisfying micro-interaction.
  • Rocket header — Consistent across all states. Gives a "mission" feel without being over the top.
  • Persona recap — Compact at top. Shows what was configured. If nothing was configured (skipped), shows warning.
  • State 3 (scanning) — Brief transition screen before auto-navigating to Home dashboard. Shows initial progress.
